# Loom theme review

Each proof sheet uses the same vault, viewport, and six surfaces:

1. Graph
2. Inbox
3. Board / Council
4. Thread details
5. Appearance settings
6. Provider settings

The provider work was also reviewed separately at the same narrow viewport:

![Codex and OpenRouter connection controls](screenshots/provider-connections.jpg)

The final picker contains six themes. The review pass also raises tertiary text
and semantic success/warning/danger colors to at least WCAG AA contrast against
every app surface.

## Typography presets

Typography is selectable independently from the color theme. Loom bundles the
custom faces locally, so no Google Fonts request is required at runtime.

1. Loom Editorial — Fraunces + Inter + JetBrains Mono
2. Source Classic — Source Sans 3 + Source Serif 4 + Source Code Pro
3. Modern — Manrope throughout + JetBrains Mono
4. Literary — Newsreader + Manrope + JetBrains Mono
5. Hyperlegible — Atkinson Hyperlegible Next + JetBrains Mono
6. System Native — operating-system sans, serif, and mono faces

![Typography preset picker](screenshots/font-presets.jpg)

## Final six-theme set

![Final six-theme picker](screenshots/final-theme-picker.jpg)

| Theme        | Score | Place in the set                                         |
| ------------ | ----: | -------------------------------------------------------- |
| Midnight Ink |   9.5 | Canonical dark; strongest hierarchy and Loom identity    |
| Paper        |   9.4 | Canonical warm light; the original Loom voice            |
| Porcelain    |   9.2 | Crisp cool light; best neutral workspace option          |
| Herbarium    |   9.1 | Natural archival light; most distinctive new light theme |
| Lagoon       |   9.0 | Expressive dark; strongest graph color separation        |
| Ember        |   8.7 | Warm dark; provides a genuinely different nighttime mood |

## Retired-theme migration

- Foundry → Paper
- Slate → Porcelain
- Dune → Herbarium
- Carbon, Obsidian, and Nocturne → Midnight Ink
- Mulberry → Ember
